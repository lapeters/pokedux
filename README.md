## Installation

- Clone repository
- Navigate to cloned directory
- Initialize node and install dependencies like so:

```
npm install
```

## Usage

### Compiles and hot reloads for development using Webpack dev server
```
npm run start
```

### Compiles and minifies for productions
```
npm run build
```
